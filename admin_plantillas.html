Filtros para Consulta SQL
Parámetros Nombrados para Filtros SELECT SQL:
En los filtros de tipo "SELECT SQL", puedes usar parámetros nombrados para crear consultas dinámicas:
  • SELECT id, nombre FROM clientes WHERE nombre LIKE [[buscar]]
  • SELECT id FROM presupuestos WHERE anio = [[anio]] AND estado = [[estado]]
Los nombres entre [[]] deben coincidir con los nombres de otros filtros (ó usar el nombre del propio filtro).
